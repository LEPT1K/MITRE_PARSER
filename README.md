# MITRE Data Parser

Парсер для скачивания и преобразования данных из официальных источников MITRE (CAPEC, CWE, CVE, ATT&CK) в JSON с русскоязычным описанием.

## 🚀 Быстрый старт

```bash
# 1. Активировать виртуальное окружение
# Windows:
venv\Scripts\activate
# Linux/macOS:
source venv/bin/activate

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить парсер
python src/main.py
```
